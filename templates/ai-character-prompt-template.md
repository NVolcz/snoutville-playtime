# AI Character Sprite Prompt Template

Use this template when generating character sprite candidates.

## Prompt

Create a clean 2D cartoon sprite of `{CHARACTER_NAME}` for a cozy preschool pretend-play web game.

Style requirements:
- Bright, simple, friendly cartoon style
- Flat colors with soft outlines
- Clear silhouette
- Child-friendly expression
- Minimal detail
- Transparent background
- Full body visible
- Character facing slightly toward the camera
- No text
- No background scenery
- No extra objects or accessories unless specified

Character requirements:
- Character: `{CHARACTER_NAME}`
- Pose/animation state: `{ANIMATION_STATE}`
- Emotion: `{EMOTION}`
- Intended use: draggable scene character
- Keep proportions consistent with previous approved sprites

Export requirements:
- PNG
- Transparent background
- Square canvas
- Character centered horizontally
- Feet/body base near the bottom-center anchor
- Leave padding around the character so no part is cropped

Negative prompt / avoid:
- Realistic rendering
- 3D render
- Complex shadows
- Background scenery
- Text or logos
- Extra limbs
- Cropped body parts
- Inconsistent face shape
- Unwanted props
