# AI Fixed Object Sprite Prompt Template

Use this template when generating fixed scene object sprites.

## Prompt

Create a clean 2D cartoon sprite of `{OBJECT_NAME}` for a cozy preschool pretend-play web game.

Style requirements:
- Bright, simple, friendly cartoon style
- Flat colors with soft outlines
- Clear silhouette
- Minimal detail
- Transparent background
- No text
- No full background scene

Object requirements:
- Object: `{OBJECT_NAME}`
- Animation state: `{ANIMATION_STATE}`
- Intended use: fixed scene object
- Object should be readable at small mobile sizes
- Object should visually invite interaction

Export requirements:
- PNG
- Transparent background
- Square canvas
- Object centered in frame
- Leave padding so animation frames do not crop

Negative prompt / avoid:
- Realistic rendering
- 3D render
- Complex shadows
- Background scenery
- Text or logos
- Extra unrelated objects
- Cropped edges
